# WPF UI Bug Playbook

## Metadata

- Issue class: WPF UI rendering, layout, theme, or binding defect
- Load level: metadata, summary, full

## Summary

Use when a WPF UI defect is concrete but not yet narrowed to a specific control template or scrollbar issue.

## Full Rule

Collect:

1. visible symptom
2. affected control and visual tree area
3. binding errors or resource lookup failures
4. theme or style resource involvement
5. minimal reproduction and screenshot where possible
