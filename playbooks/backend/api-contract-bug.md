# API Contract Bug Playbook

## Metadata

- Issue class: API request or response contract breakage
- Load level: metadata, summary, full

## Summary

Use when a bug appears to come from a changed, undocumented, or mismatched API contract.

## Full Rule

Collect:

1. expected contract
2. actual request and response shape
3. affected producers and consumers
4. compatibility expectations
5. failing contract or integration test
